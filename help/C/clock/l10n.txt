SCREENSHOT INSTRUCTIONS
------------------------

clock_applet.png

1. By default, the Clock applet is displayed on the Menu Panel (the panel on the top edge 
   of the desktop). If the Clock is not on the panel, add it by right-clicking on
   the panel and choosing Add to Panel ->  Accessories -> Clock.
2. Take a screenshot of the desktop.
3. Use the crop tool to crop the screenshot to include only the Clock applet.
4. Save the file in the PNG format.
